# OnTravel

OnTravel é um aplicativo móvel desenvolvido com React Native que permite aos usuários gerenciar suas viagens de forma simples e intuitiva. Os usuários podem adicionar, editar e excluir suas viagens, além de registrar informações como localização, data e uma descrição.

## Tecnologias Utilizadas

- **React Native**: Framework para desenvolvimento de aplicativos móveis.
- **Expo**: Plataforma para desenvolvimento, build e deploy de aplicações React Native.
- **Firebase**: Utilizado como backend para armazenamento de dados e gerenciamento de usuários.
- **React Navigation**: Biblioteca para gerenciar a navegação entre telas.
- **expo-image-picker**: Permite aos usuários selecionar imagens da galeria ou tirar fotos com a câmera.
- **@react-native-community/datetimepicker**: Para seleção de data.
- **Geolocation**: Para acessar a localização do usuário.
- **React Native Vector Icons**: Biblioteca para usar ícones vetoriais em aplicativos React Native.
